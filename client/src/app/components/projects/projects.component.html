Hotel-Booking-Website

This project is a hotel booking website that allows users to book rooms in hotels. Users can view the available rooms, select the room they want, and book it. The website also provides information about the hotel, such as its location, amenities, and services. Users can also view reviews and ratings of the hotel before booking a room.

codeZone Social Media platform

codeZone is a social media platform for developers to connect and share their work. Users can create a profile, add posts, like and comment on posts, and follow other users. The platform also has a chat feature that allows users to communicate with each other. codeZone is built using the MERN stack and features a responsive design.
